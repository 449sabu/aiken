# Getting Started

## Installation

### From Source

```console
$ cargo install aiken
```

## QuickStart

```
$ aiken --help
```

## Editor Integrations

| Editor     | Plugin                                                                      |
| ---------- | --------------------------------------------------------------------------- |
| VSCode     | [TxPipe/vscode-aiken](https://github.com/txpipe/vscode-aiken/)              |
| Vim/Neovim | [CardanoSolutions/aiken-vim](https://github.com/CardanoSolutions/aiken-vim) |
